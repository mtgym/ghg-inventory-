# GHG Dashboard Update

## Terminology Replacement
- "Konsumsi" has been replaced with "Aktivitas" throughout the document, including tabs, labels, and options.

## New Features
1. **Total Scope 3 Option**:
   - Added a new option for Total Scope 3 in the Komparasi Bebas (Free Comparison).

2. **Cumulative KPI Cards**:
   - New KPI cards have been created to sum across selected years for Total Emisi, Scope 1, Scope 2 Location, Scope 2 Market, and Scope 3. 
   - These cumulative KPIs will utilize the global year filter to calculate totals when multiple years are selected.

---

_All changes aim to improve clarity and functional capabilities of the GHG Dashboard._
